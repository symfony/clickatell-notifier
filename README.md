Clickatell Notifier
===================

Provides [Clickatell](https://www.clickatell.com) integration for Symfony Notifier.

DSN example
-----------

```
CLICKATELL_DSN=clickatell://ACCESS_TOKEN@default?from=FROM
```

where:
 - `ACCESS_TOKEN` is your Clickatell auth access token
 - `FROM` is the sender
